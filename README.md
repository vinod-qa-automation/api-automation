# API Automation Framework

## Project Description
This project automates REST APIs using Rest Assured and TestNG.
It supports multiple user roles such as Shopper, Admin, and Merchant.

## Tech Stack
- Java
- Rest Assured
- TestNG
- Maven

## Features
- Role-based authentication
- Request and response validation
- Reusable BaseTest setup
- Endpoint separation
- TestNG execution

## Modules Covered
- Shopper Module
- Admin Profile Module
- Merchant Profile Module
- Admin Action Module

## How to Run

Run using Maven:

mvn clean test
